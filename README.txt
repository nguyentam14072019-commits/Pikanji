Kanji Pikachu Full
==================

This is a complete Android Studio project for a Pikachu-style tile matching game using 214 Kangxi radicals.

Build in Android Studio:
1. Open the folder as a project.
2. Sync Gradle.
3. Build > Build APK(s).

Notes:
- The app is offline.
- It includes classic UI, hint, shuffle, level progression, and sound beeps.
- Meaning labels are provided through a mixed dictionary plus fallback labels for obscure radicals.
